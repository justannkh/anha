# 🖤 Ankha Bot — Telegram-бот для сообщества

Многофункциональный Telegram-бот для групповых чатов с интегрированным ИИ-ассистентом, системой экономики, модерацией и геймификацией.

## Возможности

**🤖 ИИ-чат (Groq API / LLaMA 70B)**
- Персонализированные ответы в образе персонажа «Анха»
- Индивидуальные персоны для пользователей (тон, факты, ник)
- Рейт-лимит, автоочистка истории, фолбэк между моделями

**💰 Экономика**
- Баланс, ежедневные/еженедельные награды, пассивный доход за активность
- Казино: слоты (`.дэп`), рулетка, коинфлип
- Магазин: VIP/Premium/Elite-статусы, лутбоксы, буст репутации
- Переводы монет между пользователями
- Мини-игра «Работа» — 10 сценариев с рандомными исходами

**🛡 Модерация**
- Ролевая система: owner → friend → admin → moder → user
- Варны (3 = автомут), муты с таймером, баны
- Лог предупреждений, декораторы прав доступа

**💍 Браки**
- Предложения, принятие/отклонение, развод
- Топ браков по длительности
- Принудительные браки/разводы (для овнера)

**✨ Репутация**
- Слова-триггеры: респект, красавчик, +, gg, based и др.
- Кулдаун между одними и теми же юзерами
- Защита от мультиаккаунтов (лимит получений в день)

**🎭 РП-команды**
- 20+ действий с гифками: обнять, ударить, поцеловать, пнуть...
- Алиасы на русском и английском

**📺 YouTube-мониторинг**
- Автопостинг новых видео в Telegram-канал
- Фильтрация Shorts, проверка даты публикации

## Технологии

| Компонент | Технология |
|-----------|-----------|
| Фреймворк | aiogram 3.7 (asyncio) |
| ИИ | Groq API (OpenAI-совместимый) |
| База данных | SQLite + aiosqlite (WAL-режим) |
| Язык | Python 3.11+ |
| Деплой | Docker / Docker Compose |

## Быстрый старт

### Локально (без Docker)

```bash
git clone https://github.com/YOUR_USERNAME/ankha-bot.git
cd ankha-bot

python -m venv venv
source venv/bin/activate        # Linux/macOS
# venv\Scripts\activate         # Windows

pip install -r requirements.txt

cp .env.example .env
# Заполни .env своими данными

python main.py
```

### Через Docker

```bash
git clone https://github.com/YOUR_USERNAME/ankha-bot.git
cd ankha-bot

cp .env.example .env
# Заполни .env

docker compose up -d --build
```

Логи: `docker compose logs -f`
Остановка: `docker compose down`

## Получение ключей

1. **BOT_TOKEN** — создай бота у [@BotFather](https://t.me/BotFather)
2. **GROQ_API_KEY** — зарегистрируйся на [console.groq.com](https://console.groq.com) (бесплатно)
3. **OWNER_ID** — узнай свой ID у [@userinfobot](https://t.me/userinfobot)
4. **ALLOWED_GROUP_ID** — добавь бота в группу, перешли любое сообщение из группы [@userinfobot](https://t.me/userinfobot)

## Команды бота

| Команда | Описание |
|---------|----------|
| `.помощь` | Список всех команд |
| `.профиль` | Профиль пользователя |
| `.баланс` | Текущий баланс |
| `.дейли` | Ежедневная награда |
| `.дэп [ставка]` | Слот-машина |
| `.рулетка [ставка] [цвет]` | Рулетка |
| `.магазин` | Каталог товаров |
| `.брак @user` | Предложение руки |
| `.рп` | Список РП-команд |
| `.варн` | Предупреждение (модер+) |
| `.мут [время]` | Замутить (модер+) |
| `.бан` | Забанить (админ+) |
